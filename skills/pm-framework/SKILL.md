## Key PM Frameworks Included

### Jobs-to-Be-Done (JTBD)
Understand why customers hire your product:
- Job to be done
- Forces (pushes/pulls)
- Hiring/firing criteria
- Anxiety and habits

### Opportunity Solution Tree
- Outcomes over outputs
- Map opportunities
- Generate solutions
- Test with experiments

### Amazon PRFAQ
- Press-release first
- FAQ section
- Start with customer obsession

### Teresa Torres Frameworks
- Continuous discovery habits
- Problem interviews
- Solution interviews

## Without Installation

If you don't install the full framework, use these prompts:

### Write a PRD
```
Create a PRD for [feature]:
- Problem: [what problem does it solve?]
- Target user: [who is it for?]
- Success metrics: [how do we measure success?]
- Requirements: [list key requirements]
- Timeline: [expected delivery]
```

### User Story Format
```
As a [user type], I want [action], so that [benefit].

Acceptance Criteria:
- [ ] [criterion 1]
- [ ] [criterion 2]
- [ ] [criterion 3]
```

### Competitive Analysis
```
Research [competitor] and provide:
- Product overview
- Key features
- Pricing model
- Strengths
- Weaknesses
- Opportunities for us
```
